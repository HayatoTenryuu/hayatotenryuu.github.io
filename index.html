About me

I am a Mechanical Designer in the Automotive Industry using Solidworks. I enjoy turning ideas and concepts into tangible and functional products.

My job is largely in designing tooling for automotive companies, whether that be designing welding fixtures, or creating assemblies for improved logistics and ergonomics. I have experience in Finite Element Analysis using Patran/Nastran and Abaqus from my college curriculum, and I have simulation experience using Solidworks.

My programming experience was gained through personal interest via online classes from Udemy, and I have done a number of personal projects to apply and improve upon what my classes taught. The programming languages that I have studied are Python, MATLAB (which I have also had college classes on), C++/C, C#, VBA (for macros in Excel and Solidworks API), and Prolog.

My future goals are to pursue web development using HTML/CSS/Javascript (which I know enough to edit, but I cannot make from scratch currently), to learn embedded programming using C++/C, and to become more familiar with cross-language applications. The reason for the latter is because I have run across a few situations when it would be easier to use a programming language, such as Prolog to handle complex logic, in combination with another language to handle functionality.
